# 🌌 Connected Notes - AI Powered Knowledge Database

Connected Notes は、ニュース記事、Googleドキュメント、PDFなどを取り込んで自分だけの知識データベースを構築するツールです。スマホ・PCのブラウザで動作し、Gemini AIを使って内容の要約、キーワードの自動抽出、タイトルの最適化などを行います。

このガイドに従って順番に進めれば、**プログラミングの知識がなくても**自分専用のアプリとして公開・使用することができます。

---

## 🚀 主な機能

- **マルチソース取り込み**: GoogleドキュメントURL、PDF、Word、MHTML、テキストファイルに対応。
- **AI解析・要約**: Gemini AIが内容を要約し、重要なキーワードを抽出。
- **自動リンク生成**: ノート内の `[[タイトル]]` 記法により、関連するノート同士を自動でリンク。
- **AIタイトル提案**: ✦ ボタン一つで、内容に最適なタイトルをAIが生成。
- **カスタマイズ可能なプロンプト**: 要約やキーワード抽出に使用するAIへの指示（プロンプト）を自由に変更可能。
- **PWA対応**: スマホのホーム画面に追加して、アプリのように使用可能。
- **Googleスプレッドシート連携**: データを自分のスプレッドシートで安全に管理。

## 🛠 セットアップガイド

### STEP 1: データベース（Googleスプレッドシート）の作成
1. [Googleスプレッドシート](https://sheets.google.com/) を開き、「空白」を新規作成します。
2. URLから「スプレッドシートID」をコピーして控えておきます。
   - 例: `https://docs.google.com/spreadsheets/d/` **`ここがIDです`** `/edit`

### STEP 2: バックエンド（Google Apps Script）の作成
1. [Google Apps Script (GAS)](https://script.google.com/home) を開き、「新しいプロジェクト」を作成します。
2. `Code.gs` に、本リポジトリの `Code.gs` の内容をすべて貼り付けます。
3. 6行目付近の `SS_ID` を、STEP 1でコピーしたIDに書き換えます。
4. **デプロイ**: 右上の「デプロイ」→「新しいデプロイ」→「ウェブアプリ」を選択。
   - 実行ユーザー: **自分**
   - アクセスできるユーザー: **全員**
5. 発行された **ウェブアプリの URL** をコピーして控えておきます。

### STEP 3: アプリファイル (`index.html`) の設定
1. `index.html` をテキストエディタで開きます。
2. **1300行目付近** にある `API_URL` を、STEP 2のURLに書き換えます。
   ```javascript
   const API_URL = "https://script.google.com/macros/s/xxxxxxxxxx/exec";
   ```
3. 保存します。

### STEP 4: GitHub Pages での公開
1. GitHubに新しいリポジトリを作成します。
2. 以下のファイルをアップロードします：
   - `index.html`
   - `manifest.json`
   - `sw.js`
3. 「Settings」→「Pages」から、Branchを `main` に設定して保存。数分後に公開URLが表示されます。

## ✦ AI設定と使い方

### Gemini API キーの設定
1. アプリを開き、右上の「**⚙️（歯車アイコン）**」をクリックします。
2. [Google AI Studio](https://aistudio.google.com/app/apikey) で取得したAPIキーを入力して保存します。

### プロンプトのカスタマイズ
設定画面内の「**📝 プロンプト設定**」ボタンから、AIが要約やタイトルを生成する際の「指示内容」を細かく調整できます。自分の好みに合わせた要約スタイルに変更可能です。

### ✦ タイトルの再生成
ノート編集画面でタイトル入力欄の右側にある「**✦**」ボタンを押すと、現在の本文に基づいてAIが最適なタイトルを提案します。

## 📱 スマホでの利用
- **iPhone**: SafariでURLを開き、「共有」→「ホーム画面に追加」
- **Android**: ChromeでURLを開き、「設定（︙）」→「アプリをインストール」

これにより、オフライン環境でも起動可能なアプリとして快適に利用できます。
