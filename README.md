# 🌌 Connected Notes - AI Powered Knowledge Database

![Banner](connected_notes_banner.png)

**Connected Notes** は、Wikiリンク、ナレッジグラフ、そして最先端のAI（Gemini）を融合させた、次世代のパーソナル・ナレッジ・マネジメント（PKM）ツールです。ニュース記事、Googleドキュメント、個人的なアイデアを「つながり」として管理し、あなただけの強力な知識ベースを構築します。

---

## ✨ 主な特徴

### 🕸️ インタラクティブ・ナレッジグラフ
- **ノートモード**: `[[ノート名]]` でリンクされた記事同士の繋がりを可視化。
- **フォルダモード**: カテゴリ間の「意味的な関係性」を俯瞰。
- **フォーカス機能**: 特定のノードをクリックして、関連する知識をハイライト。

### 🤖 強力なAI（Gemini）連携
- **自動要約・キーワード抽出**: 記事の内容を瞬時に解析し、要点とタグを自動生成。
- **AIフォルダ解析**: フォルダ間の目に見えない「意味的な繋がり」をAIが発見し、グラフ上に描画。
- **差分AI解析**: 追加・更新された記事のみを効率的に解析。トークン消費を最小限に抑えつつ、知識ベースを最新の状態に保ちます。
- **タイトル最適化**: ✦ ボタン一つで、内容に最適な魅力的なタイトルをAIが提案。

### 🔒 プライバシー & セルフホスト
- **Google Sheets 連携**: データベースはあなた自身のGoogleスプレッドシート。データは完全にあなたのコントロール下にあります。
- **GitHub Pages 対応**: 静的ファイル（HTML/JS）のみで構成されているため、無料で安全に公開・利用可能。

### 📱 快適なマルチデバイス体験
- **PWA対応**: スマホのホーム画面に追加すれば、ネイティブアプリのような使い心地。
- **オフライン対応**: 通信環境が不安定な場所でも、キャッシュを利用してノートを閲覧・作成可能。

## 🛠 セットアップガイド

### 1. データベースの準備（Googleスプレッドシート）
1. [Googleスプレッドシート](https://sheets.google.com/) で新規シートを作成します。
2. URLから「スプレッドシートID」をコピーします。
   - 例: `https://docs.google.com/spreadsheets/d/` **`SPREADSHEET_ID`** `/edit`

### 2. バックエンドの構築（Google Apps Script）
1. [Google Apps Script](https://script.google.com/home) で「新しいプロジェクト」を作成します。
2. 本リポジトリの `Code.gs` の内容をコピー＆ペーストします。
3. `const SS_ID = "..."` の部分を、先ほどコピーしたシートIDに書き換えます。
4. 「デプロイ」→「新しいデプロイ」→「ウェブアプリ」を選択。
   - 実行ユーザー: **自分**
   - アクセスできるユーザー: **全員**（認証はAPIキーで行います）
5. 発行された **ウェブアプリの URL** を控えておきます。

### 3. アプリの設定 (`index.html`)
1. `index.html` を開き、`const API_URL = "..."` を先ほどのウェブアプリURLに書き換えます。
2. GitHubリポジトリに `index.html`, `manifest.json`, `sw.js`, `connected_notes_banner.png` をアップロードします。
3. 「Settings」→「Pages」から公開設定を行います。

## ✦ AIの使いこなし方

1. **APIキーの設定**: 右上の⚙️アイコンから Gemini API キー（Google AI Studioで無料取得可能）を入力します。
2. **AI解析**: ノートを開き、「✦ AI解析」ボタンを押すと、要約とキーワードが自動入力されます。
3. **グラフ解析**: グラフ画面（フォルダモード）で「✦ AIフォルダ解析」を押すと、フォルダ間の意味的なリンクが生成されます。
   - *ヒント*: 記事が増えた後は「差分解析」ボタンを使うことで、新しい記事のみを対象に賢く更新できます。

- **Backend**: Google Apps Script (GAS)
- **Database**: Google Sheets
- **AI**: Google Gemini API (1.5 Flash / 2.0 Flash)
